# Ecommerce Sales & Customer Analysis (SQL)

## Overview
This project analyzes ecommerce transaction data using SQL to uncover insights about sales performance, customer behavior, and product trends.

## Dataset
The analysis uses three tables:

- **customers**
- **products**
- **orders**

## Key SQL Concepts
- **Joins**
- **Aggregations (SUM, AVG, COUNT)**
- **GROUP BY & HAVING**
- **Date functions**
- **Window functions (RANK)**

## Questions Answered
The project answers **20 business questions** across:

- **Data Exploration**
- **Sales Analysis**
- **Customer Analysis**
- **Product Performance**

## Tools
- **PostgreSQL**
- **SQL**

## Key Insight
The analysis identifies revenue trends, top customers, and product performance to support **data-driven business decisions**
