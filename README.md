# E-commerce Sales & Customer Analysis (SQL)

## Project Overview
This project analyzes e-commerce transactional data using SQL to understand customer purchasing behavior and sales performance.

The goal is to extract meaningful insights from the database using SQL queries that evaluate revenue trends, customer activity, and product performance.

---

## Business Questions

The analysis addresses the following questions:

- What is the total sales generated by the business?
- Which products contribute the most revenue?
- Who are the top customers based on purchase value?
- How frequently do customers place orders?
- What patterns exist in customer purchasing behavior?

---

## Tools Used

**SQL**
- Data querying
- Aggregation functions
- Filtering and grouping
- Data exploration

---

## Analysis Approach

The analysis was performed through structured SQL queries that included:

- Aggregating total sales
- Identifying top-performing products
- Ranking high-value customers
- Evaluating order frequency
- Analyzing revenue trends

---

## Key Insights

- A small percentage of customers contribute significantly to total revenue.
- Certain products consistently generate higher sales.
- Order frequency varies across different customer groups.
- SQL queries allow efficient extraction of business insights from large datasets.
